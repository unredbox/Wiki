Hardware Monitoring

These applications are helpful to know the state of your kiosk's PC.

CrystalDiskInfo

Program is used to determine the health of your HardDrive/Solid State Drive.

Features Include:

  • S.M.A.R.T. Health Monitoring
  • Temperature Tracking
  • Power-On Hours & Usage Statistics
  • SSD Endurance & Wear Leveling (NAND Health)
  • Firmware & Interface Details
  • Alerting & Logging
  • Advanced APM/AAM Control (HardDrive Only)
  • NVME Support (In Windows 10 PC or any PC replacement)
  • Portable & Low Overhead

HWInfo64

Program can be used to review the hardware configuration of a system, CPU/GPU/Memory specs, as well as monitor almost all sensor data available. Does more than HWMonitor on its own. May have a time limit when using certain features without a commercial license. Cannot be used commercially without a license.

Features Include:

  • Real-time sensor monitoring with graphing, running averages, easy resetting
  • Multi-column sensor data, allowing for widescreen, efficient sensor monitoring
  • Sensor support for motherboard, chipset, accessory sensor info
  • Supports all CPUs, GPU, Memory sensors
  • Report function for dumping the system's hardware configuration in various formats
  • Memory timings information
  • CPU / GPU feature breakdown, detailed specs
  • Remote Monitoring (12hr limit without pro)

HWMonitor

Program is used to monitor CPU, Memory (RAM), GPU (If installed), SSD/HDD, Battery, sensors, and more.

Features Include:

  • Real-time sensor monitoring
  • CPU & GPU on-die sensor support
  • LPCIO chip monitoring
  • Memory-module thermal sensors
  • SSD/HDD S.M.A.R.T. temperature monitoring
  • Battery status tracking (Requires TrippLite to have a working battery)
  • Complete report export
  • Minimal overhead & portable mode
  • Multi-platform support (will work on Windows 7 Embedded version of the kiosk software)

Features Requires a Pro License:

  • Remote Monitoring
  • Graph Generator
  • Improved Interface
